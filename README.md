Footy Chrome Extension
========================

## Quick-Start

 * Download [_footy-ext.crx](dist/_footy-ext.crx) (follow link and click `Raw`)
 * Goto `chrome://extensions/`
 * Drag and Drop file into Chrome 

## Build Locally

To create the extension:
 * `npm i`
 * `npm run build` 
 * the folder `footy-ext` can no be used in dev mode
